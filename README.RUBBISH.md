### 环境
nodejs v8.x +
python 2.7.x 

### 主要技术框架

#### 后端-backend

+ koa --- nodejs web 开发框架
+ koa-router --- 基于koa的路由框架
+ puppeteer --- google官方的无界面chrome 工具包
+ nodejieba --- 中文分词工具包，底层依赖c++和python
+ nodemailer --- nodejs的邮件发送模块包

### 前端-frontend

+ bootstrap --- 前端ui库
+ vue --- 现代web前端开发框架
+ axios --- ajax请求库
+ g2 --- 阿里开源图表库
